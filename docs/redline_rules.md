# PMO Red-Line Rules

Only prioritize these three lines first:

1. Schedule red line
- any issue that impacts master milestones must escalate immediately

2. Profit red line
- any item that may create loss must trigger warning and action owner

3. Collection red line
- any receivable risk gets elevated priority

Rule of focus:

If an issue does not affect schedule/profit/collection, do not let it steal PMO primary attention.
